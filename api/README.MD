小笨自采集自适应壁纸网站源码
========
基于孟坤壁纸二次开发

壁纸种类多、自动更新，自动采集、瀑布流加载、全站 ajax!

壁纸采集自 360壁纸库、必应首页的每日图片以及金山词霸开放平台

相关api见 [https://zhouxiaoben.info/?p=9794](https://zhouxiaoben.info/?p=9794) 

本作品禁止任何形式的倒卖，转载请注明出处！
